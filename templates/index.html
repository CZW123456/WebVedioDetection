About this project
A video object detection project that takes the video which is uploaded by the user and does object detection based on Yolo v3. The pre-trained weights for Yolo v3 is utilized

How to use
Upload a video file selecting in the file browser and click the "Upload" button. The video will be uploaded to the server and a new web page will be presented. In the new page, two options are prepared for the user——you can either start the vedio detection or selected another vedio of interest. Once you selected to start the processing, the server will run object detection using Yolo v3 model. Hopefully this procedure will not cost you too much time if the vedio size is moderate. The original vedio and the processed vedio will be shown in the website and you can view the result then.

How it works
  • Once you upload a video file, it extract the raw vedio stream in the uploaded vedio using ffmpeg-python, which is a Python ffmpeg wrapper.
  • After getting some meta information of the vedio, it opens an Video Stream where one can extract arbitrary bytes from the stream. To speed up the process, instead of processing a single image each time, we extract a batch of frames and perform object detection on the extracted frames each time to exploit all computing power of the backhaul GPUs and reduce computing latency.
  • We also extract the audio of the uploaded vedio by using Pydub which is a python audio processing library
  • You can download the processed vedio
Note
  • We used the pre-trained weights for Yolo v3, the user should download them first using the script we provide in the repo.
  • The vedio coding scheme for the uploaded vedio must be H.264 otherwise the uploaded vedio may not be successfully presented in the browser.
